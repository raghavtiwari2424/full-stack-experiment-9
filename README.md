This repository contains experiments related to **Docker, CI/CD, and Cloud Deployment** in modern full-stack development.

### 🔹 9.1 — Docker React Deployment

* Containerization using Docker
* NGINX configuration for serving React apps
* Creating Docker images
* Running applications inside containers

---

### 🔹 9.2 — CI/CD using GitHub Actions

* Automating build and deployment
* Writing workflow using YAML
* Continuous Integration setup
* Continuous Deployment pipeline

---

### 🔹 9.3 — AWS Deployment

* Deploying applications on AWS
* Cloud hosting concepts
* Server setup and configuration
* Scalable deployment practices
